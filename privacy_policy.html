Introduction

To facilitate compliance with the Chrome Web Store Developer Program Policies, this privacy policy outlines how user data is collected, stored, and used by our extension. We are committed to protecting your privacy and ensuring transparency in our practices.

Single Purpose

The extension is designed solely to provide software testers with a centralized list of tools categorized by testing type (manual, automation, API, security, performance). Users can easily access the relevant tool links directly from the extension.

Permission Justification

  • Permission Used: "storage"
  • Justification: The "storage" permission is required to save user preferences, such as favorite categories or tools, for a more personalized experience. No additional permissions are requested beyond the extension's purpose.

Data Usage

The extension does not collect any user data beyond what is necessary to fulfill its single purpose. Specifically:

  • No personally identifiable information, health information, financial information, or authentication data is collected.
  • No web history, user activity, or website content is tracked or stored.

We certify that:

  • We do not sell or transfer user data to third parties, outside of the approved use cases.
  • We do not use or transfer user data for purposes that are unrelated to our extension's single purpose.
  • We do not use or transfer user data to determine creditworthiness or for lending purposes.

Remote Code

All JavaScript files and resources are included within the extension's package. The extension does not use external scripts, modules, or dynamically evaluate code through methods like eval().
